💡 产品定义

碎片时间里的深度思考。灵感火花让你在排队、等地铁、睡前刷手机的几分钟里，完成一次完整的创意碰撞——从卡片滑动到基因杂交，从苏格拉底追问到进化树可视化。

🎯

目标用户

好奇心驱动的知识工作者、学生、创作者——那些在碎片时间里不想刷短视频、想要真正「想点什么」的人

🔑

核心价值

把「灵光一现」系统化：滑动→记录→追问→杂交→进化，一条完整的创意生长链路

📱

平台

移动优先（Flutter），Demo 可运行在任意浏览器；后端 Python FastAPI，AI 引擎可插拔

⚡ 核心功能

🎴

主题灵感卡

6 大创意主题（解压玩具、AI 工具、社交新方式…），拖动滑动卡片，左滑跳过、右滑收藏、上滑杂交

🧬

基因杂交引擎

选择两颗种子，AI 杂交生成新创意。模拟生物学杂交——两个不同领域的想法碰撞出第三种可能

❓

苏格拉底追问

对齐每个灵感进行连环追问：为什么？有什么前提假设？如果反过来呢？——把模糊想法逼成清晰概念

🌳

进化树可视化

所有灵感形成一棵树：原始想法是根，杂交后代是分支。点击任意节点回溯灵感演化路径

🌱

种子工作台

每个收藏的灵感都有独立工作台——追问记录、父系溯源、成熟度评估、一键杂交入口

⚡

创意竞技场

对战模式（二选一投票）+ 每周挑战（社区共同命题）。从个人创意到社交碰撞

❓ FAQ

灵感火花和 ChatGPT 有什么区别？

ChatGPT 是通用对话工具，对话结束就没了。灵感火花是结构化的创意生长系统——灵感有来源、有追问记录、有杂交后代、有进化路径。你不只是「聊了一个创意」，你「养了一棵创意树」。

Demo 和成品版有什么区别？

Demo 是单文件 HTML，使用 mock 数据模拟交互流程，目标是验证用户体验。成品版使用 Flutter + FastAPI + 真实 AI 模型，支持用户注册和数据持久化。

什么时候能上线？

Demo 已完成并部署。MVP 开发时间取决于资源分配，当前优先验证产品方向。有兴趣合作或内测？联系作者。

开源吗？

Demo 前端代码 ideaspark-page 公开。完整 Flutter + 后端源码在私有仓库。
